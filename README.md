# 浏览器中的拼图游戏
- 因为现有的拼图游戏支持的块数普遍偏少，并且不符合我自己的操作习惯
- 所以我编写了此项目
- 完善中
## 优势
1. 可以支持任意数量的拼图块
2. 可自定义优化级别，以适配不同性能的机器
3. 可自定义操作逻辑`完善中`
## 启动
#### 安装依赖
```
npm install
```
#### 启动项目
```
npm run dev
```
#### 构建
```
npm run build
```
